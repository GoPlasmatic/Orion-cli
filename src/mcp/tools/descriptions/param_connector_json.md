JSON string of the connector definition. Required fields: "name" (string), "connector_type" ("http" or "kafka"), "config" (object with "type" matching connector_type). HTTP config: {"type": "http", "url": "https://...", "headers": {...}, "auth": {"type": "bearer|basic|apikey", ...}, "retry": {"max_retries": 3, "retry_delay_ms": 1000}, "max_response_size": 10485760}. Kafka config: {"type": "kafka", "brokers": ["host:9092"], "topic": "...", "group_id": "..."}. Auth types — bearer: {"type":"bearer","token":"..."}, basic: {"type":"basic","username":"...","password":"..."}, apikey: {"type":"apikey","header":"X-API-Key","key":"..."}. See the connectors_create tool description for full format details and examples.
